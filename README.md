# AI Content Generator Pro

A modern AI-powered content generator built with HTML, CSS, and JavaScript.

## Features

- Advanced UI for selecting content type, output style, tone, brand voice, keywords, audience, and length
- SEO optimization and call-to-action toggles for higher-converting copy
- Live prompt preview so you can see the exact AI command before generation
- OpenAI API integration for premium output when configured
- Offline fallback content generator for instant drafts without a key
- Copy-to-clipboard, download as TXT, and save-to-history support
- Responsive design with polished glassmorphism styling

## Getting Started

1. Open `index.html` in your browser.
2. Fill out the form and click **Generate Content**.
3. Optionally enter an OpenAI API key for higher-quality results.

## OpenAI API Setup

To use the OpenAI integration:

1. Create an OpenAI account at https://platform.openai.com.
2. Generate an API key.
3. Paste the key into the **OpenAI API Key** field.

> Note: For security, do not save your key in public repositories. This demo uses the key only in the browser session.

## Files

- `index.html` — Application structure and user interface.
- `styles.css` — Visual design and responsive layout.
- `script.js` — Content generation logic, history, and OpenAI integration.

## Tips for Pro Use

- Use clear keywords and audience descriptions.
- Add custom prompt instructions for brand-specific voice.
- Save outputs to history and copy the best results.
